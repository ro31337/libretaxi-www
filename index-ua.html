Альтернатива Uber, Яндекс.таксі для пошуку водіїв та пасажирів без посередників.

LibreTaxi робить таксі знов доступним, прибираючи прошарок корпорацій з ланцюжка водій-пасажир. Пасажири сплачують поїздки готівкою і зв'язуються з водієм напряму. LibreTaxi виконує роль легкої сполучної ланки, дозволяючи домовитись щодо ціни на поїздку. Без реєстрацій, менше обмежень.

Водіям безкоштовно

Ми не стягуємо комісію. Всі гроші йдуть вам. Безкоштовно для водіїв та пасажирів.

Тільки готівка

Без кредитних карт. Сплата готівкою напряму водієві. В майбутньому інтеграція з BitCoin.

Швидка активація

Добре це чи погано - без підтверджень та перевірок. Станьте водієм за 1 хвилину!

Торгуйтесь

Вигідно для водіїв та пасажирів. Дозволяє водіям встановити ціну чи запропонувати знижку.

Всі платформи

Для айфонів, андроїда, макбука чи веб. Ми використовуємо додаток Telegram в якості транспорта.

Дух свободи

та дим вітчизни солодкий та приємний для нас. Відкритий вихідний код, менше шпигувань та обмежень.

Легко та зручно

Економно для батареї, без постійного GPS. Простий дизайн у вигляді чат-бота, мінімальне споживання трафіка.

Багатомовність

Підтримує українську, російську, англійську, індонезійську, португальську, іспанську. Допомогти з перекладом.
